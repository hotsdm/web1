HTML

다음은 습득한 HTML 태그를 나열한 것이다.

  1. Strong : 줄 굵게 만들기
  2. u : 밑줄 긋기
  3. br : 한줄 띄우기
  4. p : 단락 띄우기 / p style="margin-top:30px"를 통해 단락 상하단에 여백 조정 가능
  5. h1~5 : page의 제목 설정 / 1이 가장 큰 글자임
  6. a href : 링크 설정 태그 / href 뒤에 target을 통해 새창 띄우기 설정 / title로 링크의 제목 설정
  7. li : 목록 설정을 위한 태그 (넘버링 혹은 리스트) / ol, ul의 자식 태그
  8. ol : ordered list의 약자로 숫자 목록을 만들기 위한 li의 부모 태그
  9. ul : unordered list의 약자로 단순 목록을 만들기 위한 li의 부모 태그
  10. title : 해당 html page의 제목을 나타내는 태그
  11. meta chrset : 해당 html page를 읽는 문자 방식에 대해 나타내는 태그 (meta chrset="utf-8")
  12. body : 본문을 나타내는 태그
  13. head : 본문을 설명해주는 태그 (제목, 문자 방식 등)
  14. html : head, body를 감싸는 태그
  15. !doctype html : html로서 만들어졌다는 걸 나타내는 관용적 태그
